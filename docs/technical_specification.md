1. Общие сведения

Наименование проекта:
Система веб-форума для обсуждения игровых тем.

Основание для разработки:
Задание на производственную практику ПМ.02
(Осуществление интеграции программных модулей)

Цель проекта:
Разработка веб-приложения с API-интерфейсом и базой данных PostgreSQL для организации обмена сообщениями между пользователями.

2. Назначение разработки

Приложение предназначено для:

  - регистрации пользователей

  - создания игровых тем

  - публикации сообщений

  - просмотра списка тем и сообщений

  - хранения данных в СУБД PostgreSQL

  - обеспечения интеграции модулей через REST API

3. Требования к системе
3.1 Функциональные требования

Система должна обеспечивать:

  - Создание пользователя (POST)

  - Получение списка пользователей (GET)

  - Создание темы (POST)

  - Получение списка тем (GET)

  - Создание сообщения в теме (POST)

  - Получение сообщений по теме (GET)

3.2 Требования к базе данных

База данных должна содержать не менее 3 таблиц.

Минимальная структура:

  Таблица users

      id (PK)

      username

      email

      created_at

  Таблица topics

      id (PK)

      title

      author_id (FK → users.id)

      created_at

  Таблица posts

      id (PK)

      topic_id (FK → topics.id)

      author_id (FK → users.id)

      content

      created_at

Связи:

Один пользователь может создавать много тем

Одна тема содержит много сообщений
